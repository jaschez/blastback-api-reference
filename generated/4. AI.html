Behavior Tree based AI

The Blastback Engine makes use of behavior trees (BTs) as an entry point to provide entities with automatic behaviors.

AI properties are applied to entities via the AI Component.

To define the structure of a BT we use nodes that feed the tree with basic behaviors.

Behavior Tree Node Structure

{
	"type": "...",
	"commands": [...],
	"children": {
		...
	},
	"parameters": {
		...
	}
}

Defines a single node from the behavior tree

Fields

Name Type Description
type string
commands string[]
children Dictionary { string, Blastback.Source.Models.AI.BTNodeModel }
parameters Dictionary { string, Object }
